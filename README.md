# 🎨 FastPaint

Minimalistic, modern Paint-like drawing app built with **Python + PySide6 (Qt 6)**.

Clean desktop UI. Smooth drawing. Transparent canvas support.  
Fast, lightweight, and fully functional.

---

## ✨ Features

- 🖌 Brush & Eraser  
- 📏 Line, Rectangle, Ellipse  
- 🪣 Flood Fill  
- 🎯 Eyedropper  
- ↩ Undo / Redo (history stack)  
- 🔍 Zoom (Ctrl + Mouse Wheel)  
- 📋 Copy / Paste  
- 💾 Open / Save (PNG, JPG, BMP)  
- 🧩 Transparent canvas support  
- 🖥 Modern light UI (Fusion-based theme)

---

## 🚀 Installation

```bash
pip install PySide6
```

Run:

```bash
python paint.py
```

---

## 🛠 Tech Stack

- Python 3.13+
- PySide6 (Qt 6)
- Pure QPainter rendering
